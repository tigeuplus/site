# 매도

종목을 매도 요청 합니다.

<br/>

## 사용법

```
/매도 수량:[매도 요청할 수량] 가격:[매도 요청할 가격]
```

가격은 수량 1기준으로 수량이 0.1이라면, 가격은 0.1배 됩니다.

또한, 명령어를 사룡하려면 거래 가능 채널에서 사용해야 합니다.

<br/>
